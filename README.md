# UR5 MoveIt2 + Gazebo Simulation (ur_yt_sim)

This package contains a minimal ROS2 MoveIt2 setup found in the LeanRoboticswithROS tutorial, for simulating and controlling a UR5 robot in Gazebo using servoing and motion planning.

---

##  Features
- UR5 robot simulation in Gazebo
- MoveIt2 motion planning
- MoveIt Servo 


---

## 📦 Requirements
- ROS2 Humble
- MoveIt2
- Gazebo (Ignition or Classic depending on setup)
- Universal Robots ROS2 packages
